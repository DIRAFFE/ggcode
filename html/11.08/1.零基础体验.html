自测问题

  1. 说说你理解的 HTML 是什么？

    2. 一种标签型语言，用来写静态页面

  2. HTML 元素标签、属性都是什么概念？

    3. HTML页面内容需要都需要写在标签内；属性就是给标签添加格式、标记等各种扩展功能

  3. 文档类型是什么概念，起什么作用？

    4. 写在开头的doctype，让浏览器能读出来下面是啥东西

  4. meta标签都用来做什么的？

    5. 放在head标签内，本身没含义，用来给页面定义整体的描述或属性

  5. HTML 文档的结构组成是怎样的？

    6. 声明+html+head+body

  6. title里面的内容会显示出来吗？

    7. 放在head标签内，在页面不显示，显示在浏览器标签页上的标题。
